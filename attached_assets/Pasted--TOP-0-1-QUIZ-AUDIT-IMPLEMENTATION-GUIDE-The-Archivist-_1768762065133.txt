### **FIX #4: EDGE CASE HANDLING (All Neutral Answers)**

**Issue:** Selecting “Neutral” for all answers causes NaN error.

**Solution:** Add fallback logic for edge cases.

**File:** `/src/lib/quiz/calculatePattern.ts` (or similar)

```javascript
function calculateDominantPattern(answers) {
  const scores = {
    disappearing: 0,
    apology_loop: 0,
    testing: 0,
    attraction_to_harm: 0,
    compliment_deflection: 0,
    draining_bond: 0,
    success_sabotage: 0,
  };
  
  // Calculate scores
  answers.forEach(answer => {
    if (answer.pattern && answer.weight) {
      scores[answer.pattern] += answer.weight;
    }
  });
  
  // Find max score
  const maxScore = Math.max(...Object.values(scores));
  
  // EDGE CASE 1: All neutral (maxScore = 0)
  if (maxScore === 0) {
    return {
      pattern_id: 'neutral',
      pattern_name: 'Neutral Assessment',
      pattern_description: 'Your responses show no dominant pattern currently. This could mean:\n\n- You\'re in transition between patterns\n- Your nervous system is regulating well right now\n- You need more specific context\n\nTake the assessment again after observing your reactions in real situations.',
      body_signature: 'Varied or minimal activation',
      circuit_break: 'Observe your body\'s response in triggering situations, then retake assessment.',
      cta_text: 'Start 7-Day Crash Course',
      cta_url: '/crash-course',
    };
  }
  
  // EDGE CASE 2: Multiple patterns tied
  const topPatterns = Object.keys(scores).filter(
    key => scores[key] === maxScore
  );
  
  if (topPatterns.length > 1) {
    return {
      pattern_id: 'hybrid',
      pattern_name: 'Multi-Pattern System',
      pattern_description: `Your nervous system runs ${topPatterns.length} patterns simultaneously: ${topPatterns.map(p => patterns[p].pattern_name).join(', ')}.\n\nThis is your layered protection mechanism. Each activates in specific contexts.`,
      body_signature: 'Context-dependent pattern switching',
      circuit_break: 'Identify which pattern runs in which context: work vs relationships vs family.',
      primary_patterns: topPatterns.map(p => patterns[p]),
      cta_text: 'Get Full Pattern Map',
      cta_url: '/pricing',
    };
  }
  
  // NORMAL CASE: Single dominant pattern
  const dominantPattern = topPatterns[0];
  return patterns[dominantPattern];
}
```

**Test Edge Cases:**

- [ ] All neutral → Shows “Neutral Assessment” result
- [ ] All strongly agree → Shows dominant pattern
- [ ] Two patterns tied → Shows “Multi-Pattern” result
- [ ] Random answers → Shows dominant pattern
- [ ] Skip question → Prevented or handles gracefully

**Issue:** Might not be accessible to all users.

**Keyboard Navigation:**

```javascript
useEffect(() => {
  const handleKeyPress = (e) => {
    // Numbers 1-5 select answers
    if (e.key >= '1' && e.key <= '5') {
      const answerIndex = parseInt(e.key) - 1;
      selectAnswer(answers[answerIndex]);
    }
    
    // Enter advances to next question
    if (e.key === 'Enter' && selectedAnswer) {
      goToNextQuestion();
    }
    
    // Escape closes modal
    if (e.key === 'Escape') {
      closeQuiz();
    }
  };
  
  window.addEventListener('keydown', handleKeyPress);
  return () => window.removeEventListener('keydown', handleKeyPress);
}, [selectedAnswer]);
```
