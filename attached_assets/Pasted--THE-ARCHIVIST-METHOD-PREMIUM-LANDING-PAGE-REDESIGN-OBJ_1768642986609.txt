## BRAND DESIGN SYSTEM

### **Colors:**
- **Background Dark:** #0a0a0a (pure black) or #1a1a1a (dark gray)
- **Secondary Dark:** #111111 (section alternates)
- **Teal (Primary):** #14B8A6 - 70% of accent usage
- **Pink (Accent):** #EC4899 - 30% of accent usage
- **White:** #ffffff (headlines)
- **Light Gray:** #e5e5e5 or #f5f5f5 (body text)
- **Medium Gray:** #9ca3af (supporting text)

### **Teal/Pink Usage Rules:**
- **Teal** = Trust, method, science, primary CTAs, section headlines
- **Pink** = Energy, emotion, emphasis, hover states, premium elements
- **Always pair together** in gradients, glows, complementary placement
- **Never pink alone** - must work with teal for balance

### **Typography Scale:**
- **Hero Headline:** 64-72px desktop, 40-48px mobile, BOLD
- **Section Headlines:** 42-48px desktop, 32-36px mobile, BOLD
- **Subheadings:** 24-28px, SEMIBOLD
- **Body Text:** 18-20px, line-height 1.8-2.0
- **Small Text:** 14-16px

### **Spacing Scale:**
- **Section padding:** 120px top/bottom desktop, 80px mobile
- **Element spacing:** 40-60px between major elements
- **Card padding:** 40px internal padding
- **Card gaps:** 32px between cards

### **Visual Elements:**
- **Gothic aesthetic** - Sharp edges, not rounded
- **Library imagery** - Old books, atmospheric, mysterious
- **Particle effects** - Subtle, 70% teal / 30% pink
- **Gradient overlays** - Teal-to-pink or dark-to-transparent
- **Depth shadows** - Layered teal + pink glows

## PERFORMANCE OPTIMIZATION

### **Images:**
```
Hero background:
- WebP format with JPG fallback
- Size: 1920x1080 optimized (under 200KB)
- Lazy load: No (critical for hero)

Pattern card images (if added):
- WebP format
- Lazy load: Yes
- Size: 400x300 max
```

### **Fonts:**
```
Preload critical fonts:
- Inter Bold (headlines)
- Inter Regular (body)

Load other weights async
Use font-display: swap
```

### **Animations:**
```
Use transform and opacity (GPU accelerated)
Avoid animating: width, height, top, left (causes reflow)
Use will-change sparingly (only on hover elements)
```

### **Code:**
```
Minimize CSS (remove unused)
Defer non-critical JavaScript
Lazy load below-fold images
Use CSS containment where possible
```
