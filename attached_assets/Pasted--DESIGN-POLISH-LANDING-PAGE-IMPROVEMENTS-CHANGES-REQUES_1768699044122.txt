## 3. FLOATING PARTICLES ON ALL PAGES

### **Current State:**
Particles might only be on hero

### **New State:**
Particles on EVERY section (150 on desktop, 75 on mobile)

**Implementation:**

```html
<!-- Add particle container to body -->
<div class="floating-particles"></div>
```

```css
.floating-particles {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  pointer-events: none;
  z-index: 1;
  overflow: hidden;
}

.particle {
  position: absolute;
  width: 2px;
  height: 2px;
  background: radial-gradient(circle, rgba(20, 184, 166, 0.8) 0%, transparent 70%);
  border-radius: 50%;
  animation: float-particle 20s infinite;
  opacity: 0;
}

/* Pink particles mixed in (20% of particles) */
.particle.pink {
  background: radial-gradient(circle, rgba(236, 72, 153, 0.6) 0%, transparent 70%);
}

@keyframes float-particle {
  0% {
    transform: translateY(100vh) translateX(0) scale(0);
    opacity: 0;
  }
  10% {
    opacity: 0.6;
  }
  90% {
    opacity: 0.6;
  }
  100% {
    transform: translateY(-100vh) translateX(50px) scale(1);
    opacity: 0;
  }
}

/* Particles are generated dynamically with random delays and positioning */
/* See JavaScript implementation for particle generation */

/* Mobile - reduce particle count */
@media (max-width: 768px) {
  /* Particles reduced to 75 on mobile (handled in JS) */
}
```

```javascript
// Generate particles (MATCH CURRENT AMOUNT)
const particleContainer = document.querySelector('.floating-particles');

// Desktop: 100-150 particles
// Mobile: 50-75 particles
const particleCount = window.innerWidth > 768 ? 150 : 75;

for (let i = 0; i < particleCount; i++) {
  const particle = document.createElement('div');
  // 20% pink, 80% teal
  particle.className = i % 5 === 0 ? 'particle pink' : 'particle';
  
  // Random positioning
  particle.style.left = Math.random() * 100 + '%';
  particle.style.animationDelay = Math.random() * 20 + 's';
  particle.style.animationDuration = (15 + Math.random() * 10) + 's';
  
  // Random size variation
  const size = 1 + Math.random() * 2; // 1-3px
  particle.style.width = size + 'px';
  particle.style.height = size + 'px';
  
  particleContainer.appendChild(particle);
}
```

**Effect:**
- **150 particles on desktop** (visible, atmospheric)
- **75 particles on mobile** (still visible, not overwhelming)
- 20% pink, 80% teal mix
- Random positioning and timing (organic feel)
- Size variation (1-3px)
- Works on ALL pages (fixed position)
- Premium, alive atmosphere

## 4. FIX TEXT ANIMATIONS (WEB + MOBILE)

### **Current Issue:**
Archival Reveal animation might not work on mobile

### **Fix:**

**Desktop Animation (Keep):**
```css
.brand-title {
  transform: translateY(100%);
  opacity: 0;
  animation: archival-reveal 1.2s cubic-bezier(0.77, 0, 0.175, 1) 0.3s forwards;
}

.brand-tagline {
  opacity: 0;
  animation: fade-in 0.8s ease-out 1.0s forwards;
}

@keyframes archival-reveal {
  from {
    transform: translateY(100%);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}

@keyframes fade-in {
  from {
    opacity: 0;
    transform: translateY(5px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**Mobile Fix (Simpler Animation):**
```css
@media (max-width: 768px) {
  .brand-title,
  .brand-tagline {
    /* Simpler fade-in on mobile (more reliable) */
    animation: mobile-fade-in 0.8s ease-out forwards;
  }
  
  @keyframes mobile-fade-in {
    from {
      opacity: 0;
      transform: scale(0.95);
    }
    to {
      opacity: 1;
      transform: scale(1);
    }
  }
}
```

**Reduced Motion Accessibility:**
```css
@media (prefers-reduced-motion: reduce) {
  .brand-title,
  .brand-tagline,
  .pattern-card,
  .particle {
    animation: none !important;
    transform: none !important;
    opacity: 1 !important;
  }
}
```

**Pattern Card Animations (Fix):**
```css
.pattern-card {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.6s ease-out, transform 0.6s ease-out;
}

.pattern-card.visible {
  opacity: 1;
  transform: translateY(0);
}

/* Stagger (already there, keep) */
.pattern-card:nth-child(1) { transition-delay: 0s; }
.pattern-card:nth-child(2) { transition-delay: 0.1s; }
.pattern-card:nth-child(3) { transition-delay: 0.2s; }
/* etc... */

/* Mobile - reduce stagger */
@media (max-width: 768px) {
  .pattern-card:nth-child(n) {
    transition-delay: 0s !important;
  }
}
```

## 5. KEEP PREMIUM FEEL (BALANCE PINK)

### **Pink Usage Rules:**

**DO use pink for:**
- ✅ Strategic word emphasis in headlines
- ✅ "Not" in tagline
- ✅ Pattern card numbers (accent)
- ✅ CTA button hover (glow)
- ✅ 20% of floating particles
- ✅ "FREE" in pricing

**DON'T use pink for:**
- ❌ Body text
- ❌ All headlines (just key words)
- ❌ Button backgrounds (keep teal)
- ❌ Borders (keep #333)
- ❌ Pattern card backgrounds

**Balance:**
- Teal = 70% (primary brand color)
- Pink = 20% (strategic accents)
- White/Gray = 10% (text)
