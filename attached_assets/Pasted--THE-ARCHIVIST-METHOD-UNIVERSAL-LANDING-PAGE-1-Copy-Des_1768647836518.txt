## RESPONSIVE DESIGN (Mobile Strategy)

### **Mobile Breakpoints:**
```css
/* Tablet */
@media (max-width: 1024px) {
  /* 2-column grids become 1-column */
  /* Reduce padding by 20% */
  /* Font sizes reduce by 10% */
}

/* Mobile */
@media (max-width: 768px) {
  /* All grids become 1-column */
  /* Reduce padding by 40% */
  /* Font sizes reduce by 20% */
  /* Buttons full-width (90%) */
  /* Navigation becomes hamburger */
}
```

### **Mobile-Specific Optimizations:**

**Hero:**
- Background image more zoomed, darker overlay
- Brand lockup 20% smaller but still prominent
- Headline 40-48px (down from 56-64px)
- Button full-width with max-width 90%
- Padding reduced to 60px top

**Pattern Cards:**
- Single column, full width
- 20px side margins
- Padding reduced to 24px
- Touch-friendly (no hover dependency)
- Slightly larger tap targets

**Pricing Cards:**
- Stack vertically
- Full width with 20px margins
- Maintain visual hierarchy (free still emphasized)
- Buttons full-width

**Navigation:**
- Hamburger menu icon (24x24px)
- Full-screen overlay when open
- Large touch targets (min 44px height)
- Close X in top right

## PERFORMANCE OPTIMIZATION

### **Critical Path:**
1. Above-fold HTML/CSS loads first
2. Hero image lazy loads (but prioritized)
3. Below-fold content lazy loads
4. Non-critical JS deferred

### **Image Strategy:**
```
Hero background:
- Format: WebP with JPG fallback
- Size: 1920x1080, optimized <200KB
- Preload: Yes (critical)

Pattern card images (if added):
- Format: WebP
- Size: 400x300 max
- Lazy load: Yes
```

### **Font Loading:**
```html
<link rel="preload" href="/fonts/Inter-Bold.woff2" as="font" crossorigin>
<link rel="preload" href="/fonts/Inter-Regular.woff2" as="font" crossorigin>
```

### **Animation Performance:**
```css
/* Only animate transform and opacity (GPU accelerated) */
.animate {
  transform: translateY(0);
  opacity: 1;
  transition: transform 0.3s ease, opacity 0.3s ease;
  will-change: transform; /* Use sparingly */
}
```
