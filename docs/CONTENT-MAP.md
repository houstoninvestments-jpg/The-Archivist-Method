## Overview

The canonical book lives at `the-archivist-method/`. It is the voice source for every other surface in the ecosystem. PDFs and product variants compile from this directory; the source of truth is always the markdown.

---

## Canonical Book — `the-archivist-method/`

```
the-archivist-method/
├── module-0-emergency/                  # 4 files — crisis triage
├── module-1-foundation/                 # 6 files — what this is, why not therapy, who it isn't for
├── module-1a-threshold/                 # 1 file  — threshold reading frame
├── module-2-four-doors/                 # 5 files — Four Doors framework + each door
├── module-2-5-demonstration/            # 1 file  — worked demonstration of the protocol
├── module-3-patterns/                   # index + 9 pattern directories × 12 files = 109 files
├── module-4-implementation/             # 9 files — 90-day map, weekly phases, daily practice, what this book cannot do
├── module-5-advanced/                   # 3 files — multiple patterns, combinations, relapse protocol
├── module-6-context/                    # 4 files — work, relationships, parenting, body
├── module-7-field-notes/                # 1 file  — letters from the field
├── module-8-resources/                  # 5 files — reading, professional help, finding a therapist, supporting someone, glossary
├── epilogue/                            # 1 file  — epilogue
└── back-matter/                         # 5 files — three-tier framing, ecosystem doorway, Pocket Archivist trial, recommender appendix, closing page
```

The compiled book lives at `builds/the-archivist-method-FULL.md` (markdown) and `builds/the-archivist-method-FULL.pdf` (PDF). Both are regenerated from the canonical markdown — never edit the build outputs directly.

## Derived Product PDFs

These are the products generated from the canonical book.

### The Complete Archive PDF

- **Generator:** `scripts/generate_complete_archive.py`
- **Output:** `outputs/THE-ARCHIVIST-METHOD-COMPLETE-ARCHIVE.pdf`
- **Source:** all canonical chapters in `the-archivist-method/`
- **Structure:** parts + epilogue, with pull quotes inserted for visual rhythm

### The Crash Course PDF (free tier discovery)

- **Generator:** `scripts/generate_crash_course.py`
- **Output:** `outputs/THE-ARCHIVIST-METHOD-CRASH-COURSE.pdf`
- **Content:** self-contained inside the script (not assembled from canonical files)
- **Structure:** Title → What This Is → Nine Patterns overview → Day 1–7 → What's Next
