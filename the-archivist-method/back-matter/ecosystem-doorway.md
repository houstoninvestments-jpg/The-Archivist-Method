# IF YOU WANT TO KEEP GOING

The book ends here. The work does not.

Some readers close the back cover and the pattern continues to run, less often, less intensely, until one day they realize they did not run it this week. That happens. It is the trajectory the framework was built to produce.

Other readers want a longer corridor. They want the framework continued. They want a tool that names which pattern is theirs without ambiguity. They want a daily companion that stays in the pocket and reminds them, in the gap, what to do.

That is what the rest of the ecosystem is for.

The first door is the quiz. Two minutes. Nine patterns. It identifies which of the nine you are running primarily, with secondary patterns ranked. The result is a starting point that this book cannot give you, because the book had to address all readers running all patterns. The quiz addresses only you.

The quiz is at:

**thearchivistmethod.com/quiz**

It is free. There is nothing to install. No account required to receive the result. Take it on a phone, on a laptop, in a waiting room.

After the quiz, the next page in the ecosystem describes what happens next. Read that page or do not. Either way, the quiz is the first concrete thing you can do with what you have learned in these pages.

The Archivist sees you across the threshold.
